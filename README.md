# scriptStore

脚本仓库

> 用于记录一些自动化脚本操作

### 脚本列表

| 序号 |   脚本名称    |                      具体功能                      |
| :--: | :-----------: | :------------------------------------------------: |
|  1   | buildTime.js  |            打印 Taro 项目构建时的时间戳            |
|  2   | dealFiles.js  | 遍历指定路径下的文件，对文件内容处理后，再写入文件 |
|  3   | openVSCode.sh |  快速打开 VSCode（可配合 Sourcetree 自定义使用）   |
|  4   | psdLittle.js  |                    PS 文件瘦身                     |
|  5   | switchGit.sh  |                 切换 git 账号脚本                  |
